Visualising the data

Let’s generate some plots of the data. There are many ways to plot data in R. One of the most popular is to use the ggplot2 package. This will be discussed in more detail in lectures and tutorials.

To plot your data using ggplot2, you specify three things:

  1. Your data
  2. Your aesthetics (for example, what’s on the x-axis)
  3. Your layers (for example, barplot, scatterplot)

The first argument is the data you’re visualising. Next you define the aesthetic mappings. In other words, the columns of the data that get mapped to certain aesthetic features of the plot, e.g. the x axis will represent the variable called x and the y axis will represent the variable called y. Note that the aes() function is really a way to map the aesthetics of the plot to the variables in the data. See the Modern Dive book (by Chester Ismay & Albert Y. Kim) metioned on the course Blackboard page for more details.

Then we add another layer to this plot where we define which geometric shapes we want to use to represent each observation in the data.

If this seems like a lot, it is. Don’t panic, you will learn about the philosophy of building data visualizations in more detail over the course and in future assignments. For now, follow along with the code that is provided.

First you’ll plot a boxplot of the life expectancy of each continent. The code is ‘read’ as follows: create a plot using the gapminder data , use continent as the aesthetic for the x-axis and life expectancy as the aesthetic for the y-axis, and and the shape I want to plot is a boxplot with certain labels for each axis.

ggplot(gapminder, aes(x = continent, y = lifeExp)) +
  geom_boxplot() +
  ylab("Life Expectancy (years)") +
  xlab("Continent")

extra bonus assignment

If you would like to try an extra task we will create a lineplot to see how life expectancy has changed over time for any chosen country. The first thing you need to do is to first filter out the data for the country in question and then pass this data into the plotting function.

You can read the code like a sentence: “start with the gapminder dataframe and then filter out all rows that contain Ireland in the country variable, then create a plot by mapping year to the x axis and life expectancy to the y axis, display all the data for a country in the same line and use a different colour for each country, adding a line on as the layer of the plot.” Note that the ‘%>%’ command is read as ‘then’ in the sequence of arguments while the ‘+’ is used by ggplot to add new components to the graph.

gapminder %>% filter(country %in% c("Ireland")) %>% 
  ggplot(aes(x = year, y = lifeExp, group=country, color=country)) +
  geom_line() +
  ylab("Life Expectancy (years)") +
  xlab("Continent")

To compare more than two counries (e.g. Ireland and Rwanda) add in the extra country (exactly as spelt in the dataset) as follows:

gapminder %>% filter(country %in% c("Ireland", "Rwanda")) %>% 
  ggplot(aes(x = year, y = lifeExp, group=country, color=country)) +
  geom_line() +
  ylab("Life Expectancy (years)") +
  xlab("Continent")
